# ☕ SQL Coffee Shop Inventory Management System

## 📌 Project Overview
The Coffee Shop Inventory Management System is a SQL-based mini project developed to manage and analyze coffee shop operations efficiently. The project stores information related to products, customers, orders, sales, and inventory using relational database concepts.

This project demonstrates practical implementation of SQL queries including JOINs, GROUP BY, aggregate functions, subqueries, filtering, and table relationships.

---

# 🚀 Features

- Product Management
- Customer Order Tracking
- Inventory Monitoring
- Sales Analysis
- Revenue Calculation
- Top Selling Products
- Customer Purchase Details
- SQL JOIN Operations
- Aggregate Function Queries

---

# 🛠️ Technologies Used

- SQL
- MySQL
- Relational Database Management System (RDBMS)

---

# 🗂️ Database Tables

| Table Name | Description                 |
|------------|-----------------------------|
| Customers  | Stores customer details     |
| Orders     | Stores order information    |
| Items      | Stores coffee shop products |
| Inventory  | Tracks product stock        |
| Sales      | Stores sales records        |
